1. Discuss the scope of var, let, and const ?

Answer : Let and Const are block scoped, var globally scoped.Let variables can be reassign or updated but not re-declared, Const variables can neithe be updated nor re-declared and var variables can be updated and re-declared.

2.Tell us the use cases of null and undefined ?

Answer : Null that means an blank value or empty value.Undefined that means the variables has been declared.But not values set or assigned.The typeof operator returns undefined for an variables. The typeof operator returns the type as an object for a variable.

3.What do you mean by REST API ?

Answer : API that means Application Programming Interface, is set rules that define how applications ord devices can connect to and communicate with each other. A REST API is an API that conforms to the design principles of the REST.
